# [CPE 查榜小工具](https://tzufu.me/CPE) 

CPE 查榜小工具 
使用者可輸入姓名查詢 CPE 排名、題數、用時及排名比例，並可依學校或單位篩選結果。  
> 由於官方網站顯示的排名與爬蟲抓取的資料不同，本工具使用自行計算的排名與排名比例，若有錯誤或異常，歡迎回報。

**資料來源**： [CPE 官方網站](https://cpe.mcu.edu.tw/)  

**技術說明**：  
- **資料收集**：Python 爬蟲  
- **前端**：HTML、CSS 與 JavaScript 建立查詢介面
